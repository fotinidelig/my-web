# Netlify Quick Start Checklist

## Before First Deploy

- [ ] Push code to Git repository (GitHub/GitLab/Bitbucket)
- [ ] Verify `netlify.toml` is in root directory
- [ ] Test build locally: `npm run build`

## Netlify Setup (5 minutes)

1. **Connect Repository**
   - Go to netlify.com → "Add new site" → "Import project"
   - Connect Git provider → Select `my-web` repo
   - Build settings should auto-detect (verify: `npm run build`, `dist`)

2. **Add Custom Domain**
   - Site settings → Domain management → Add `byfotini.com`
   - Follow DNS instructions (use Netlify DNS or add A/CNAME records)

3. **Deploy**
   - First deploy happens automatically
   - Wait for DNS propagation (up to 48 hours)
   - SSL certificate auto-provisions (1-2 hours after DNS)

## Verify After Deploy

- [ ] Site loads at `byfotini.com`
- [ ] HTTPS is active (green lock icon)
- [ ] Test security headers: https://securityheaders.com/?q=https://byfotini.com
- [ ] Check build logs for any warnings

## Cost Monitoring

- **Free tier**: 300 build minutes/month (~100-150 builds)
- **Monitor**: Site settings → Usage
- **Optimization**: Already configured in `netlify.toml`
