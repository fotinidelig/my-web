# Personal Portfolio (Astro + React + Tailwind)

## Run locally

```bash
npm install
npm run dev
```

- Dev server: http://localhost:4321
- Build: `npm run build`
- Preview: `npm run preview`

## Edit content
- Projects: `src/data/projects.json`
- Skills: `src/data/skills.json`
- Gallery: `src/data/gallery.json`
- Section copy: `src/components/*.astro`

## Structure
- Layout: `src/layouts/BaseLayout.astro`
- Pages: `src/pages/index.astro`, `src/pages/gallery.astro`
- Components: `src/components/*`
- Styles: `src/styles/global.css`

## Notes
- Theme toggle stores preference in `localStorage`.
- Tailwind `darkMode: 'class'`.
- Font: Cabin via Google Fonts.
